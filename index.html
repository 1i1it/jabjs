    JabJS

    JabJS allows you to bind any JavaScript model to any DOM element only JavaScript, via the following idiomatic syntax:

    jab.bind(model, 'property', domElement[s]);

    For example:

    user = {name: 'John Lennon'}; //any JavaScript object
jab.bind(user, 'name', document.getElementByid('input')); //user is now binded (two-way) with #input

    Now the model and the element are binded together (synced). Changing either will change the other. This includes native assignment on the model's side: 

    user.name = 'George' //#input.value is now 'George'

    Features:

    • Bind any JS model to any DOM element (or to other JS variables) with a one-liner.
    • Stand-alone & dependency-free, simply include and run.
    • Tiny: ~1.5K compressed.
    • Pure JS, creates bindings without changing HTML markup: Keep your JS out of your markup.
    • Source code is easily readable and modifiable - understand exactly what is going on.
    • Customizable bindings for your own needs.
    • Orthogonal to other JS libraries - use it anywhere, with any other library or templating, without depending or modifying anything else.
